## 🧩 기술 스택

-   Node.js (NestJS)
-   TypeORM
-   PostgreSQL / MySQL
-   Jest (테스트)
-   Swagger (API 문서)

---

## 📆 개발 기간

-   2025.05.31 \~ 2025.06.20

## 📌 기능 요약

-   사용자 잔액 충전 / 조회
-   상품 조회 및 주문 처리
-   선착순 쿠폰 발급 및 적용
-   인기 상품 통계 조회
-   외부 데이터 연동 (Mock 기반)
